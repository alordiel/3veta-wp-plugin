== Description ==
"3veta Booking Page for WordPress" is an official plugin maintained by the 3veta team. It allows you to embed your 3veta booking page to your WordPress website in a simple and easy way.
Through this plugin, visitors can schedule meetings directly on your WordPress website. In this way, you can have a fully white-label booking page!

== Installation ==
=
From your WordPress dashboard
=
1. Visit 'Plugins > Add New'
1. Search for '3veta Booking Page for WordPress'
1. Activate 3veta Booking Page for WordPress from your Plugins page. To start building your dropdown go to Settings -> Dropdown multisite.
= From WordPress.org
=
1. Download '3veta Booking Page for WordPress'.
1. Upload the '3veta' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, rsync, etc...)
1. Activate it from your Plugins' page.
